# Lab 13 - Mobile Navigation

## Описание
В данной лабораторной работе реализована мобильная навигация в React Native с использованием библиотеки React Navigation.

## Что реализовано
- Stack Navigation
- Bottom Tab Navigation
- Переходы между экранами
- Передача параметров через route params
- Иконки вкладок
- Badge для уведомлений
- Экран настроек
- Экран поиска
- Экран уведомлений
- Экран профиля

## Структура проекта
- App.tsx
- src/navigation/AppNavigator.tsx
- src/navigation/types.ts
- src/screens/HomeScreen.tsx
- src/screens/ProfileScreen.tsx
- src/screens/SettingsScreen.tsx
- src/screens/SearchScreen.tsx
- src/screens/NotificationsScreen.tsx
